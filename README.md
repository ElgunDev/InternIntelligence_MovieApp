# Movie App

Technologies Used

- Kotlin 
- MVVM & Clean Architecture 
- Dagger Hilt – Dependency injection
- Firebase Firestore & Authentication 
- Retrofit & okhttp3:logging-interceptor – TMDB API
- Glide – Şəkil yükləmə
- ViewPager – Sürüşdürmə və keçidlər
